# 作品名：Bear

## 概要
YouTubeのチュートリアルを参考に制作した、スタジオライティングのクマの3Dモデリング作品

## シーン構成
- 3Dモデル (クマ・グリーンバック)
- camera
- right (キーライト(point right)・トップライト(area right))

## 工夫点：ライティング
本作品では、プロのスタジオ撮影を意識したライティングを構築した。
| ライト配置前 | ライト配置後 |
| :---: | :---: |
| ![配置前](readme-images/Bear_before.png) | ![配置後](readme-images/Bear_after.png) |

特にライティングに注力し、以下の演出を行った。
1. 視線誘導のための「キーライト」
クマの顔に焦点を当てるため、ポイントライトを前方に配置した。
瞳にキャッチライトが入るよう調整し、キャラクターの表情を生き生きとさせた。

2. 実在感を高める「エリアライト」
頭上からエリアライトを照射することで、足元に柔らかな影を生成した。
これによりモデルが空間に浮いているような違和感を消し、存在感を表現した。

3. スタジオセットの構築（グリーンバック）
現実のモデル撮影と同様に背後に曲面の平面メッシュを配置し、グリーンバック環境を作成した。
被写体と背景を明確に分離し、モデルのシルエットを際立たせている。

## 使用技術
- モデリング（サブディビジョンサーフェイス）
- テクスチャリング（PBRマテリアル）
- ライティング・レンダリング（Cycles）

# 作品名：Gun

## 概要

本作品は、人気FPSゲーム『VALORANT』に登場する武器スキン「ネオフロンティア シェリフ（レベル4 / ヴァリアント1 ゴールド）」をモチーフにした3Dモデリング作品である。近未来的なデザインと、リボルバーが持つクラシックな雰囲気が融合した独特のデザインを再現している。

## シーン構成

- 3Dモデル (銃・フリー画像)
- camera
- light (メインライト(spot light)・アクセントライト(point light))

## 工夫点：モデリング

- 忠実なフォルムの再現
シェリフ特有の重さを感じるシルエットを崩さず、各パーツの形状を細部まで忠実に再現した。
- アニメーションを実現するパーツ構成

## 工夫点：ライティング

モデルの持つ「紫色の世界観」と「金属の高級感」を強調するため、以下のライト配置を行った。
| ライト配置前 | ライト配置後 |
| :---: | :---: |
| ![配置前](readme-images/Gun_before.png) | ![配置後](readme-images/Gun_after.png) |


- 発光エフェクト
　シリンダーや銃身の発光部にはポイントライトを配置し、内側から光が漏れ出す表現により、近未来感を演出した。
- 雰囲気作り
    
    　スポットライトを正面から当てることで、全体を明るく見せつつ、背景と馴染む幻想的な紫色の空気感を構築した。
    
- 質感の強調
　金属パーツの質感を出すため、3つのポイントライトを金属付近に配置した。ゴールド部分に鋭いハイライトを入れることで、硬質感・金属感を強調している。

## 使用技術

- モデリング（サブディビジョンサーフェイス）
- テクスチャリング（PBRマテリアル）
- ライティング・レンダリング（Eevee）
